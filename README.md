# Supervised Regression Examples in Python

- This repository contains simple and didactic Python examples of supervised regression models, developed for educational purposes.

- The goal is to demonstrate, in a clear and concise way, how different regression techniques work using small and easy-to-understand scripts.

## Models Included

- **Simple Linear Regression**
  - Demonstrates the relationship between a continuous independent variable and a continuous target variable.
  - Focuses on understanding model fitting and interpretation.

- **Logistic Regression**
  - Demonstrates a classification problem using a supervised learning approach.
  - Focuses on probability estimation and binary decision boundaries.

## Purpose of the Repository

- Educational and instructional use
- Demonstration of supervised learning concepts
- Introductory examples for students in Data Science, Statistics, or Machine Learning

## Technologies Used

- Python
- NumPy
- Pandas
- Scikit-learn
- Matplotlib (for visualization, if applicable)
